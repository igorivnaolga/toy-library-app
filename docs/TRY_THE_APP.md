# Try the app

Install the **Church Corner Toy Library** app on your **Android phone**. You do **not** need a computer, a GitHub account, or any technical setup — only the link or file the library sends you.

## What you need

- An **Android** phone (the pilot is Android-only for now).
- The **download link** or **APK file** from the library (GitHub Release, email, WhatsApp, etc.).
- An **email address** to sign in or register in the app.

## Download the app

You only need the file named something like **`app-release.apk`**. You do **not** need to download source code or clone anything from GitHub.

### From a GitHub Release link

If the library sent you a link to a **Releases** page on GitHub:

1. Open the link **on your Android phone** in **Chrome** (or another browser).
2. Scroll down to **Assets** (below the release notes).
3. Tap **`app-release.apk`** (or a similar `.apk` file).
4. If Chrome warns about the file type, tap **Download anyway** (or **Keep**).
5. Wait for the download to finish — continue with [Install on your phone](#install-on-your-phone) below.

You do **not** need a GitHub account to download the APK from a public release.

### Other ways to get the file

| How the library shared it | What to do |
|---------------------------|------------|
| **Google Drive / Dropbox** | Open the link on your phone → download the APK file. |
| **Email or chat** | Save the attachment, or tap it to download. |

## Install on your phone

1. When the download finishes, open **Files**, **Downloads**, or your notification and tap **`app-release.apk`**.
2. If Android asks whether you trust this app, tap **Settings** (or **Allow**) and turn on **Install unknown apps** / **Allow from this source** for **Chrome**, **Drive**, or **Files** — only for that app.
3. Tap **Install**, then **Open**.

**First time only:** you may see a welcome screen with building blocks — tap **GET STARTED**.

## Sign in or join

1. **Already a member?** Tap **Sign in** and use your email and password.
2. **New to the library?** Tap **Join the library** (or register) and follow the steps.

Opening hours, location, and contact details are in the app under **Contact** (or **Info**).

## Updating to a newer version

When the library publishes an update, open the **same Releases page** (or new link they send you), download the **latest** **`app-release.apk`**, and install again. You may install over the existing app; your account and data stay the same.

## Troubleshooting

| Problem | Try this |
|---------|----------|
| **Install blocked** | Allow installs from the app you used to download (Chrome, Drive, Files). |
| **Can’t sign in** | Check email/password; use **Forgot password** if needed. |
| **App won’t load toys** | Check mobile data or Wi‑Fi; try again in a few minutes. |
| **Still stuck** | Contact the toy library — use the details in the app or on the library website. |
